# CU04 - Buscar Viaje

**Actor:** Pasajero

**Objetivo:** Permitir al usuario consultar viajes disponibles según origen y destino.

## Guion

1. El pasajero abre “Buscar viajes”.
2. Ingresa origen y/o destino.
3. El sistema muestra coincidencias en estado DISPONIBLE.
4. El pasajero selecciona un viaje para ver detalles.

## Excepciones

### **E1 – No existen viajes**
1. El sistema muestra: “No hay viajes disponibles”.

## Postcondiciones
- El usuario visualiza viajes disponibles.
