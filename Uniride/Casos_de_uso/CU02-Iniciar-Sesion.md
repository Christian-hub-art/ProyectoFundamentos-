# CU02 - Iniciar Sesión

**Actor:** Estudiante registrado

**Objetivo:** Permitir que un usuario entre a su cuenta.

## Guion

1. El actor ingresa su correo institucional.
2. El actor ingresa su contraseña.
3. El sistema valida credenciales.
4. El sistema inicia la sesión del usuario.

## Excepciones

### **E1 – Credenciales incorrectas**
1. El sistema muestra “Correo o contraseña incorrectos”.
2. El usuario puede reintentar.

## Postcondiciones
- El usuario queda autenticado en el sistema.
